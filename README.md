NodeJs-OnlineShop is pet project created with Node.js, MongoDB and EJS.
It's a simple online shop where as a user you can check products info, add to cart, create orders and so on.

## Installation

Use the package manager [npm]([https://pip.pypa.io/en/stable/](https://www.npmjs.com/)) to install all dependencies.

```bash
npm install
```

## Run

```python
# run app in development mode 
npm run start

```
